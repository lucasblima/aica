## Key Features

### Memory Categories
- **profile:** Static user info (name, location, timezone)
- **preference:** User choices (communication_style, theme)
- **fact:** Observed truths (work_hours, emotional_triggers)
- **insight:** AI conclusions (correlations, patterns)
- **pattern:** Behavioral trends (productivity_peaks)

### Security
- Row-Level Security enforces user isolation
- JWT-based user_id extraction
- Service role bypass for automation
- GDPR-compliant deletion

### Performance
- 10 optimized indexes
- GIN index for JSONB search
- Expected latency <50ms for most queries
