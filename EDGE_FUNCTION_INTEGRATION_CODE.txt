SE ALGO DER ERRADO
==============================================================================

Erro: "Cannot find module"
  → Verificar path: "./daily-question-handler.ts" (sem leading slash)

Erro: "Action desconhecida"
  → Verificar se case foi adicionado corretamente no switch
  → Verificar se action é exatamente "generate_daily_question"

Erro: "Timeout ou resposta vazia"
  → Normal, significa fallback para Level 2/3
  → Verificar gemini_api_logs table no Supabase

Erro: "Invalid payload"
  → Verificar se userContext tem estrutura correta
  → Verificar se systemPrompt e contextSummary são strings
