================================================================================
                         ARCHITECTURE PATTERN
================================================================================

SECURITY DEFINER FUNCTIONS

Problem: RLS policies cannot directly query tables (infinite recursion 42P17)

Solution: Create SECURITY DEFINER functions that:
  1. Bypass RLS (only functions, not policies)
  2. Execute as superuser in controlled context
  3. Have SET search_path = public for security
  4. Return boolean for policy decisions

Functions Created:
  - is_connection_space_member(_space_id uuid)
  - is_connection_space_admin(_space_id uuid)
  - is_connection_space_owner(_space_id uuid)

Benefits:
  DONE Prevents infinite recursion
  DONE Single source of truth for permissions
  DONE Professional security pattern
  DONE Easier to audit
  DONE Easier to maintain
